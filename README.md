# VIYF 更新镜像源

## 简介

[VIYF][viyf] 为 NVDA 中文社区用户免费提供 [NVDA 更新镜像源][viyfMirror]，可提高 NVDA 的更新速度，有效提升用户体验。

用户只需安装本插件，随后按正常流程执行“检查更新”或等待软件自动更新，当有新版本发布时则会检测到更新并引导用户下载。
